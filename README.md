## Vue H5模板

开箱即用。


## 使用方式

``` shell
# clone the project
git clone https://github.com/erdong0604/vue-h5-template.git

# enter the project directory
cd vue-h5-template

# install dependency
yarn or npm install

# develop
yarn serve or npm run serve
```
